# 🎓 Complete MATLAB Course for Absolute Beginners

**Welcome!** This course assumes you have **ZERO** coding knowledge. Every lesson is written
in plain English, explains every word of jargon, and gives you code you can copy, run,
and play with.

---

## 🛠️ What you need before starting

You need MATLAB installed on your computer. You have 3 options:

| Option | Cost | Link |
|--------|------|------|
| **MATLAB Online** (easiest — runs in your browser, nothing to install) | Free basic version (20 hrs/month) | https://matlab.mathworks.com |
| **MATLAB Home / Student license** | Paid (student version is cheap) | https://www.mathworks.com/store |
| **GNU Octave** (free clone of MATLAB — 95% of this course works in it) | 100% Free | https://octave.org |

👉 **Recommendation for you:** Start with **MATLAB Online** (free). Just create a free
MathWorks account and you can code in your browser immediately.

## 🗺️ How to study (a plan that works)

1. **One lesson per day.** Don't rush. 30–45 minutes daily beats 5 hours once a week.
2. **Type the code yourself.** Don't just read — typing builds muscle memory.
3. **Break things on purpose.** Change numbers, remove characters, see what errors appear.
   Errors are teachers, not enemies.
4. **Do the exercises** before looking at solutions.
5. After lesson 10, **redo the final project from memory**. If you can, you've truly learned it.

Happy coding! 🚀
